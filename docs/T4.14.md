# Task T4.14 – Add creature affection system (level, training effect)

## What was found

1. **Field existed but no mechanics**: The `CreatureInstance` type in `src/types/game.ts` already had an `affection: number` field (line 104), but no gameplay mechanics utilized it.

2. **Affection values set statically**: Affection was set to hardcoded values when creatures were acquired:
   - Boss egg hatchlings: affection = 5 (playerModule.ts:627)
   - Captured creatures: affection = 10 (missionModule.ts:432)
   - Fused creatures: affection = 5 (missionModule.ts:690)
   - But there was no mechanism for affection to change.

3. **No affection-based benefits**: There was no damage bonus, XP boost, capture bonus, or other effects tied to creature affection levels.

## What was done

### 1. Created `src/core/affection.ts` - Affection system core logic

- **`getAffectionLevel(affection)`** — Maps raw affection value (0-100+) to discrete levels (1-5) and provides:
  - Damage multiplier per level: 1.0x, 1.07x, 1.15x, 1.25x, 1.4x
  - XP bonus multiplier per level: 1.0x, 1.1x, 1.2x, 1.35x, 1.5x

- **`applyAffectionGain(creature, source)`** — Increases affection based on interaction type:
  - Victory in combat: +8
  - Creature training: +5
  - Successful capture: +10
  - Maximum cap: 100

- **`getAffectionDamageMultiplier(creature)`** — Returns damage multiplier based on affection level.

- **`getAffectionXPBonus(creature)`** — Returns XP bonus multiplier based on affection level.

### 2. Integrated affection into combat system (`src/stores/game/modules/combatModule.ts`)

- Updated `attackWithCreature` to apply affection damage multiplier.
- Updated `useSkill` to apply affection damage multiplier.
- Updated `handleVictory` to grant +8 affection to creatures that participated in victory.

### 3. Integrated affection into creature training (`src/stores/game/modules/missionModule.ts` and `playerModule.ts`)

- Updated `finishActivity` for `creature_training` type to grant +5 affection after training completes.

### 4. Integrated affection into mission XP system (`src/core/xpCurve.ts`)

- Updated `grantPartyXP` to apply affection XP bonus to creature XP gains during mission completion.

### 5. Added comprehensive test suite (`src/test/affection.test.ts`)

- Tests for `getAffectionLevel` covering all 5 levels.
- Tests for damage multiplier values per level.
- Tests for XP bonus values per level.
- Tests for `applyAffectionGain` with various sources.
- Tests for affection cap at 100.
- Tests verifying affection level boundaries.

## Behavioral Changes

| Creature Affection Level | Range | Damage Bonus | XP Bonus |
|---|---|---|---|
| Level 1 (Distant) | 0-19 | 1.0x | 1.0x |
| Level 2 (Neutral) | 20-39 | 1.07x (+7%) | 1.1x (+10%) |
| Level 3 (Friendly) | 40-59 | 1.15x (+15%) | 1.2x (+20%) |
| Level 4 (Bonded) | 60-79 | 1.25x (+25%) | 1.35x (+35%) |
| Level 5 (Devoted) | 80-100 | 1.4x (+40%) | 1.5x (+50%) |

### Affection Gain Sources

| Source | Amount | Condition |
|---|---|---|
| Creature victory in combat | +8 | Creature survives and wins |
| Creature training activity | +5 | Training completes successfully |
| Successful capture | +10 | Creature joins player's party |

### Formula

```
Affection Level = floor(affection / 20) + 1 (capped at level 5)
Damage Multiplier = 1 + (level * 0.07)² ≈ 1.0x, 1.07x, 1.15x, 1.25x, 1.4x
XP Bonus Multiplier = 1 + (level * 0.1) + floor(level/2) * 0.05 ≈ 1.0x, 1.1x, 1.2x, 1.35x, 1.5x
```
