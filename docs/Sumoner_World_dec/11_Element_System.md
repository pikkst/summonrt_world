# Element System

## Primary Elements
- Fire
- Water
- Earth
- Air
- Lightning
- Iron
- Nature
- Ice
- Light
- Darkness

## Character Affinity
Each character receives one primary affinity.
Rare rolls:
- Dual: 1 / 1,000
- Triple: 1 / 1,000,000
- Omni: Quest only

## Design Rules
- Elements affect capture.
- Elements affect fusion.
- Elements affect combat.
- Elements affect crafting.
- Elements affect exploration.

## Future Expansion
- Element mastery
- Element corruption
- Element resonance
- World elemental events
