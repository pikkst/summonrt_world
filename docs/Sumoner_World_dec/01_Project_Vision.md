# SummonerWorld – Project Vision

## Vision
SummonerWorld is a browser-based text RPG focused on deep simulation rather than graphics.

### Core Goals
- 100 persistent worlds
- Creature collection and fusion
- Living economy
- Dynamic ecosystem
- Offline-first architecture
- MMO-ready architecture

## Design Pillars
1. Player Freedom
2. Meaningful Progression
3. Living World
4. Procedural Generation
5. Long-term Replayability
6. AI-friendly architecture
