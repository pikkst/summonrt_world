# Creature Fusion

## Goal
Fusion is one of the game's core progression systems.

## Fusion Requirements
- Both creatures Level >= 5
- Soul Crystal
- Compatible elements

## Fusion Pipeline
1. Validate parents
2. Calculate rarity
3. Merge abilities
4. Merge traits
5. Generate appearance
6. Create offspring

## Future Systems
- DNA inheritance
- Hidden genes
- Rare mutations
- Fusion achievements
- Legendary recipes
