# Coding Standards

## Folder Structure
src/
 core/
 systems/
 ui/
 data/
 services/
 types/
 tests/

## Principles
- SOLID
- DRY
- KISS
- Data-driven design
- Event-driven architecture

## Naming
Classes: PascalCase
Functions: camelCase
Constants: UPPER_CASE
Files: PascalCase for components, camelCase for helpers

## Testing
Every gameplay system requires:
- Unit tests
- Integration tests
- Regression tests
