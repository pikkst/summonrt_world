# Sprint 0 – Task T0.4 Documentation

## Objective
Create project-level lint scripts: `npm run lint`, `npm run lint:fix`, and verify `npm run test:watch` functionality.

## Changes Made

### 1. Created `eslint.config.js`
Configured ESLint with TypeScript and React support:
- **Parser**: `@typescript-eslint/parser` for TypeScript syntax
- **Plugins**: `@typescript-eslint/eslint-plugin`, `eslint-plugin-react-hooks`, `eslint-plugin-react-refresh`
- **Globals**: Browser and Node.js globals enabled
- **Environment**: jsdom-compatible JSX parsing
- **Ignores**: `dist`, `node_modules`, and `server` directories excluded

### 2. Updated `package.json`
Added npm scripts for linting:
- `lint`: Runs ESLint without auto-fix (`eslint . --ext .ts,.tsx`)

- `test:watch`: Already existed, runs Vitest in watch mode (`vitest`)

Added ESLint devDependencies:
- `eslint@^9.0.0`
- `@eslint/js@^9.0.0`
- `@typescript-eslint/parser@^8.0.0`
- `@typescript-eslint/eslint-plugin@^8.0.0`

- `globals@^15.0.0`
