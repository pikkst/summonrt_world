# Task T5.4 – Create traitSynergy.ts: 50+ documented trait combinations

## What was found

The fusion system in `src/stores/game/modules/missionModule.ts` inherited traits from parent creatures via a simple dedup and slice approach, but there was no mechanism to combine traits for special synergy effects. Traits were passed through without any interaction logic.

The existing trait system in `src/modules/creatures/creatureFactory.ts` defined a small set of base traits:
- `regeneration` - Restores HP every turn
- `strong` - +5 Attack power
- `tough` - +3 Defense
- `swift` - +3 Speed
- `magic_affinity` - +2 Mana regeneration

However, there was no system to recognize when certain traits combine to create unique abilities.

## What was done

### 1. Created `src/data/traitSynergy.ts`

A complete trait synergy engine featuring:

- **`TRAIT_SYNERGIES` array** - Contains 50 documented trait combinations with special effects
- **Synergy key format** - Lowercase trait names joined with underscore (e.g., `regeneration_poison`)
- **Effect types supported**:
  - `statBonuses` - Adds flat stat modifications (attack, defense, speed, etc.)
  - `special` - Describes unique combat or gameplay effects
  - `elementAffinity` - Elemental damage bonuses or reductions
  - `modifier` - Additional gameplay modifiers

**Key synergies implemented include:**

| Trait Pair | Result Name | Effect |
|------------|-------------|--------|
| poison + regeneration | Acidic Recovery | Corrosive self-healing, +5 attack, acid touch on melee |
| agile + swift | Blinding Speed | +8 speed, 15% dodge chance |
| strong + might | Colossal Power | +10 attack, 2x physical damage |
| magic_affinity + arcane | Mana Overflow | +15 mana regen, +20 max mana |
| vampire + lifesteal | Bloodthirst | 25% damage dealt converts to healing |
| fire_blast + flame_body | Inferno Aura | Burning aura damages nearby enemies |
| ice_shard + frozen | Permafrost Core | +8 defense, slows enemy speed 20% |
| lightning_static + shock | Electrostatic Field | 30% chance to shock attackers |
| battle_frenzy + rage | Berserker Fury | Attack scales with missing HP |
| defender + protect | Shield Wall | 25% damage reduction for all allies |
| healer + restoration | Divine Blessing | 2x healing effectiveness |
| berserk + fury_unleashed | Critical Catastrophe | Chain critical hits |
| shield_wall + bulwark | Ultimate Defense | 70% all damage reduction |
| healing_hands + divine_smite | Purifying Light | Heals damage undead instead of healing |

## Summary

The trait synergy system now enables strategic depth in the fusion system. Players can intentionally breed creatures with complementary traits to unlock powerful combination effects, such as:

- **Defensive synergies**: Shield Wall, Ultimate Defense, Diamond Hide
- **Offensive synergies**: Colossal Power, Devastating Blow, Critical Catastrophe
- **Elemental synergies**: Inferno Aura, Permafrost Core, Electrostatic Field
- **Healing synergies**: Divine Blessing, Mana Overflow, Acidic Recovery
- **Utility synergies**: Blinding Speed, Phase Strike, Mana Mastery

When traits combine, the fused creature receives:
1. Visual indication in the nickname
2. Stat bonuses applied to base attributes
3. Special combat effects tracked in synergyEffects array
4. Log message announcing the synergy activation
