# T6.6 – Scan Ability Extension: 3-Turn Penalty & Weakness Identified Overlay

## Summary
Extended the existing "Scan" ability (T6.3) with a turn-limited damage penalty for wrong guesses and a prominent "Weakness Identified" UI overlay for correct guesses. The SCAN skill remains in the creature ability pool and the correct guess now explicitly reveals the primary elemental weakness.

## Implementation Details

### 3-Turn Penalty System
- **Wrong guess**: Sets `penaltyTurnsRemaining: 3` on `scanResult`
- **Turn decrement**: After each player combat action (`attackWithCreature` and `useSkill`), `penaltyTurnsRemaining` decreases by 1
- **Penalty expiry**: When `penaltyTurnsRemaining` reaches 0, damage multiplier returns to normal (1.0x)
- **Combat log feedback**: Shows remaining penalty turns on each attack/skill (e.g., "Wrong guess! -70% damage, 2 turns remaining")
- **Expired message**: Once penalty expires, log shows "Wrong guess penalty expired"

### Correct Guess Reveal
- On correct guess, system logs: `🔍 WEAKNESS IDENTIFIED: <element> is the primary weakness of <enemy>!`
- The guessed element is explicitly surfaced in combat log and UI

### UI Feedback
- **Correct guess overlay**: Green pulsing banner labeled "🔍 Weakness Identified" with the specific element and +30% damage confirmation
- **Wrong guess overlay**: Red pulsing banner showing remaining penalty turns during active penalty; neutral banner once expired

### Damage Calculation Update
`getGuessDamageMultiplier` now uses the turn-limited penalty:
- `guessCorrect === true` → returns `1.3` (permanent +30%)
- `guessCorrect === false && penaltyTurnsRemaining > 0` → returns `0.3` (−70% for remaining turns)
- `guessCorrect === false && penaltyTurnsRemaining === 0` → returns `1.0` (penalty expired)

## Helper Functions
