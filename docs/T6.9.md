# T6.9 – Ascending Requires Defeating Floor Guardian (or Rare Teleport Item)

## Summary

Implemented the dungeon ascension gating rule: a player must defeat the floor guardian before they can ascend to the next floor. A rare `dungeon_ascend_scroll` item can be consumed to bypass this requirement.

## Changes

### 1. `src/data/constants.ts`
- Added `DUNGEON_ASCEND_SCROLL = 'dungeon_ascend_scroll'` constant as the rare bypass item key.

### 2. `src/stores/game/modules/missionModule.ts`
- Imported `DUNGEON_ASCEND_SCROLL`.
- Modified `descendDungeon` to enforce guardian defeat requirement:
  - If `currentFloor > 0` and the floor is **not** in `clearedFloors`, descent is blocked.
  - If the player has a `dungeon_ascend_scroll` in inventory, it is consumed and ascent is allowed.
  - Otherwise, a warning log is appended: *"The floor guardian blocks your path! Defeat it to ascend, or find a rare Teleport Scroll to bypass."*
- Floor 0 (dungeon entrance lobby) remains exempt from the guardian check.

### 3. `src/stores/game/modules/playerModule.ts`
- Fixed `resolveTrapRoom`, `resolvePuzzleRoom`, and `resolveVendorRoom` to mark the current floor as cleared (`clearedFloors`) when the encounter is resolved.
- Previously only combat/victory paths and `resolveTreasureRoom` updated `clearedFloors`, which could strand the player on trap/puzzle/vendor floors under the new guardian gate.

### 4. `src/test/dungeonAscension.test.ts` (new)
- Added 6 unit tests covering:
  - First descent from floor 0 is unrestricted.
  - Un-cleared floors block descent with a warning.
  - Cleared floors allow normal descent.
  - `dungeon_ascend_scroll` consumption bypasses the guardian and removes the item from inventory.
  - Boss-floor predecessor checks behave correctly.
  - Warning message includes the Teleport Scroll hint.

## Behavior

| Scenario | Result |
|----------|--------|
| Current floor is 0 | Always allowed to descend |
| Current floor > 0, floor is cleared | Allowed to descend |
| Current floor > 0, floor is uncleared, no scroll | Blocked, warning logged |
| Current floor > 0, floor is uncleared, has scroll | Scroll consumed, descent allowed |

## Items

- **Teleport Scroll** (`dungeon_ascend_scroll`): A rare item that can be found in dungeon treasure rooms or vendors. Consumed automatically when used to bypass a floor guardian.

## Tests

All 434 tests pass (including the 6 new T6.9 tests). TypeScript compilation succeeds with zero errors.
