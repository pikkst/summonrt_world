# Task T3.4 - Implement XP Curve

## What was found
The existing getLevelThreshold function in gameEngine.ts:61-63 implemented a similar XP formula but returned a 
umber type, which would cause overflow issues for level 1000 and beyond. No dedicated XP curve module existed.

## What was done
- Created src/core/xpCurve.ts with three functions:
  - getXPThreshold(level: number): bigint - Returns XP needed to reach a specific level using formula: XP_Base = 100 × (1.15)^(Level-1)
  - getCumulativeXP(level: number): bigint - Returns total XP required to reach a level from level 1
  - getXPForLevel(startLevel, endLevel): bigint - Returns XP needed between two levels

- Created src/test/xpCurve.test.ts with tests verifying:
  - Level 1 threshold = 100 XP
  - Level 2 threshold = 115 XP
  - Level 10 threshold = 352 XP
  - Level 1000 threshold is calculable without overflow (uses BigInt)
  - Cumulative XP calculations
  - XP range calculations for level progression

- All 11 tests pass successfully using vitest
