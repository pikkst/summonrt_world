## Algorithm Details

### Rest Room Guarantee
The system ensures at least one rest room by:
1. Collecting all non-special rooms (non-treasure, non-entrance, non-boss)
2. Shuffling them with seeded random
3. Explicitly marking the first shuffled room as 'rest'

### Biome Themed Distribution
Each biome has a unique room type weight distribution that reflects the theme:
- Volcanic areas are more combat and trap-heavy
- Crystal caves emphasize puzzles
- Coastal areas have more vendors and rest rooms
- Sky islands focus on elite and puzzle rooms

### Tier-Based Scaling
Higher tier worlds (higher world index) receive modified room type weights:
- Worlds > tier 2: More traps
- Worlds > tier 1: More puzzles
- Worlds > tier 3: More elite rooms
- Worlds > tier 4: More vendor rooms
