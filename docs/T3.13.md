# T3.13 - Add Experience Display (Progress Bar) to ResourcePanel

## What was found
The `ResourcePanel` component at `src/ui/ResourcePanel.tsx` displayed four resource bars (Life, Energy, Happy, Nerve) but had no visual indication of the player's current XP progress toward the next level. The player state already tracks `level` and `experience` fields, and the XP threshold formula `XP_Base = 100 × (1.15)^(Level−1)` was defined in `src/core/xpCurve.ts`. The `getXPThreshold` function returns the XP required to reach a given level from the previous one, which maps directly to the progress bar numerator.

## What was done
Added a fifth progress bar to `ResourcePanel` for experience with the following:
- **Label**: "XP"
- **Color**: `bg-[#ffd700]` (gold) to distinguish from existing resource bars
- **Calculation**: Uses `getXPThreshold(player.level)` to determine the XP required for the next level, then computes `(player.experience / nextLevelXP) * 100` for the fill percentage
- **Import**: Added `getXPThreshold` import from `src/core/xpCurve.ts`
- **Styling**: Matches existing bar conventions (same height, border, glossy overlay, font styling, transition duration)
