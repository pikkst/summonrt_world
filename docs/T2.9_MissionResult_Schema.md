- `xp: number`

## Findings

Upon investigating the codebase (`summoner-world/src/`), the `MissionResult` schema was already fully implemented and integrated.

`summoner-world/src/types/game.ts` (lines 278–283)

### Item Type Alignment
The task requested `rewards: ItemStack[]`. The codebase uses `InventoryStack[]` as the canonical item stack representation (`src/types/game.ts`). There is no separate `ItemStack` type. The `InventoryStack` interface includes `templateKey`, `quantity`, and optional `modifiers`, matching the conceptual role of `ItemStack`.

```typescript
export interface InventoryStack {
  templateKey: string;
  quantity: number;
  modifiers?: Record<string, any>;
}
```
