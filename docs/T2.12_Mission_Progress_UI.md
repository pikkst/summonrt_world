### Features Implemented

1. **Real-time Countdown**: The component uses `setInterval` (1-second updates) to refresh remaining time for active missions.

2. **Remaining Time Display**: Format varies based on duration:
   - Hours + minutes + seconds (when hours > 0)
   - Minutes + seconds (when minutes > 0)
   - Just seconds (when under a minute)

3. **Status Badge**: Color-coded badges for each mission status:
   - `PENDING`: Gray badge
   - `IN_PROGRESS`: Indigo badge
   - `COMPLETED`: Emerald badge
   - `FAILED`: Rose/red badge
   - `CANCELLED`: Amber/yellow badge

4. **Mission Categories Display**:
   - Active Missions (with progress bar and countdown)
   - Pending Missions
   - Recently Completed Missions
   - Failed Missions
   - Cancelled Missions

5. **Mission Type Labels**: Human-readable labels for each mission type:
   - EXPLORE_TIER_1 → "Explore Sector"
   - SCOUT_DUNGEON → "Dungeon Scout"
   - SMELT_ORE → "Smelt Ore"
   - CRAFT_ITEM → "Craft Item"
   - STORE_VISIT → "Store Visit"
   - TAX_EDICT → "Tax Edict"
   - CARAVAN_ROUTE → "Caravan Route"

### UI Integration
