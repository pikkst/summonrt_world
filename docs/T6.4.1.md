## What Was Done

### dungeonGenerator.ts (new file)
Created `src/core/dungeonGenerator.ts` with the following components:

1. **getDungeonFloorSeed (lines 8-17)**:
   - Combines worldIndex, floorIndex, and globalSeed into a deterministic hash
   - Uses string hash function to produce unique seeds per floor

2. **createMaze (lines 26-62)**:
   - Implements recursive backtracking algorithm for maze generation
   - Creates a 5x5 grid of maze cells with wall boundaries
   - Uses depth-first search with backtracking to carve passages

3. **mazeToGraph (lines 112-149)**:
   - Converts maze grid into `DungeonFloorGraph` structure
   - Maps each cell to a room with connection IDs based on removed walls
   - Sets entrance at top-left (room_0_0) and boss at bottom-right

4. **assignRoomTypes (lines 151-185)**:
   - Assigns room types: entrance, boss, treasure (1-3 random), and various room types
   - Uses seeded random for deterministic room type placement

5. **generateDungeonFloor (lines 187-205)**:
   - Main entry point that orchestrates the generation pipeline
   - Calls `getDungeonFloorSeed`, `createMaze`, `mazeToGraph`, and `assignRoomTypes`
   - Returns fully populated `DungeonFloorGraph`
