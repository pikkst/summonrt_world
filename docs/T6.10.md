# T6.10 - Dungeon Exit Minimum Viable Level Scaling

## Task Description
Verify dungeon exit scales player to minimum viable level.

## Implementation Summary

### Functions Added
- `calculateMinViableLevel(worldIndex: number): number` in `src/stores/game/helpers.ts`
  - Calculates the minimum viable level required to safely exit a dungeon based on world tier
  - Formula: `max(1, 1 + floor((worldIndex - 1) * 2))`
  - World 1: level 1, World 2: level 3, World 3: level 5, World 10: level 19

- `applyMinViableLevelScaling(player: PlayerState, worldIndex: number, creatures: CreatureInstance[]): { player: PlayerState; creatures: CreatureInstance[] }` in `src/stores/game/helpers.ts`
  - Scales player stats to minimum viable level when exiting a dungeon
  - Applies to player level, life/energy max values, and creature stats
  - Preserves existing stats not affected by scaling

### Integration Points
- `fleeDungeon()` in `src/stores/game/modules/missionModule.ts`
  - Applies minimum viable level scaling on successful dungeon escape
  - Logs the scaling event to the player

- `resolveDungeonEncounter()` in `src/stores/game/modules/missionModule.ts`
  - Applies minimum viable level scaling on victory (including boss defeat)
  - Only logs scaling when player was actually scaled

### Scaling Effects
When a player exits a dungeon with level below the minimum viable level:
1. Player level is increased to the minimum viable level
2. Player life max: `100 + (level * 10)`
3. Player energy max: `100 + (level * 5)`
4. Player life/energy current restored to max
5. Each creature receives stat boosts:
   - HP: +5 per level difference (max +50)
   - Attack: +1 per level difference (max +10)
   - Defense: +1 per level difference (max +5)
   - Speed: +1 per level difference (max +5)
   - Mana: +2 per level difference

### Test Coverage

- `applyMinViableLevelScaling` no scaling when above threshold
- `applyMinViableLevelScaling` correct scaling when below threshold

- Existing player stats preserved
- Health/energy restored to max on scaling
