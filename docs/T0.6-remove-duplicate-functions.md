# T0.6 - Remove Duplicate Functions in gameStore.ts

## Task Description
Remove duplicate functions in `src/stores/gameStore.ts` (`movePlayer` vs `finishMovement`), unify API.

## Analysis

### Issue Identified
The `movePlayer` and `finishMovement` functions in `src/stores/gameStore.ts` contained duplicated logic:
1. Both calculated the same movement modifiers: `distToCenter`, `proximityFactor`, `difficultyScale`
2. Both handled neighbor tile discovery and marking them as discovered
3. Both accessed world data and tile information

### Solution Implemented

#### Extracted Helper Functions

Two helper functions were created to consolidate the duplicated logic:

**1. `calculateMovementModifiers(x, y)`**
- Location: `src/stores/gameStore.ts` (end of file, lines ~2505-2515)
- Purpose: Calculate position-based modifiers for movement cost and difficulty
- Returns: `{ distToCenter, proximityFactor, difficultyScale, energyCost }`

**2. `processTileDiscovery(x, y, currentWorldId, worlds)`**
- Location: `src/stores/gameStore.ts` (end of file, lines ~2517-2527)
- Purpose: Handle neighbor tile generation and discovery marking
- Creates tiles for neighboring coordinates if they don't exist
- Marks all neighboring tiles as discovered

#### Updated Functions

**`movePlayer(dx, dy)`**
- Now uses `calculateMovementModifiers()` to get energy cost before movement
- Still initiates exploration timer for unexplored tiles
- Directly calls `finishMovement()` for already-explored tiles

**`finishMovement(x, y, tileKey, newlyExplored)`**
- Now uses `calculateMovementModifiers()` and `processTileDiscovery()` helper functions
- Uses the extracted modifiers for consistent energy cost and difficulty calculations
- Properly handles the case where player reaches a sector but lacks energy

### Code Changes

Before:
```typescript
// In movePlayer:
const distToCenter = Math.hypot(newX - 1000, newY - 1000);
const maxDist = 1414;
const proximityFactor = Math.max(0, 1 - (distToCenter / maxDist));
const difficultyScale = 1 + proximityFactor * 4;
const energyCost = 1 + Math.floor(proximityFactor * 2);

// In finishMovement:
const distToCenter = Math.hypot(x - 1000, y - 1000);
const maxDist = 1414;
const proximityFactor = Math.max(0, 1 - (distToCenter / maxDist));
const difficultyScale = 1 + proximityFactor * 4;
// energyCost recalculated differently here
```

After:
```typescript
// Helper function:
function calculateMovementModifiers(x: number, y: number): { distToCenter: number; proximityFactor: number; difficultyScale: number; energyCost: number } {
  const distToCenter = Math.hypot(x - 1000, y - 1000);
  const maxDist = 1414;
  const proximityFactor = Math.max(0, 1 - (distToCenter / maxDist));
  const difficultyScale = 1 + proximityFactor * 4;
  const energyCost = 1 + Math.floor(proximityFactor * 2);
  return { distToCenter, proximityFactor, difficultyScale, energyCost };
}

// Both functions now use the helper
const { energyCost } = calculateMovementModifiers(newX, newY);
const { energyCost, proximityFactor, difficultyScale } = calculateMovementModifiers(x, y);
```
