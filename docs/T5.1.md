# Task T5.1 – Create fusionMatrix.ts: 10x10 element combination matrix

## What was found

The existing fusion system in `src/stores/game/modules/missionModule.ts` used inline logic to combine creature elements during breeding:

- It merged both parents' elements into a `combinedElements` array via `Set` union
- It then sliced the first 2 elements (`combinedElements.slice(0, 2)`) to form the new creature's element list
- Special mutations (`void`, `starlight`) were applied probabilistically after the fact
- There was no canonical, engine-driven element-combination lookup table; fusion results were deterministic only by array position, not by semantic pair rules

This meant the game could not enforce elemental chemistry rules such as Fire+Air=Storm or Water+Ice=Glacier without ad-hoc patches in mission logic.

## What was done

### 1. Created `src/data/fusionMatrix.ts`

- **`FUSION_MATRIX`** — A 10x10 base combination lookup (`Record<string, string>` containing 55 unique unordered pairs + 10 same-element entries)
- Each key is an alphabetically sorted `+`-delimited element pair (e.g. `'air+fire'`, `'ice+water'`)
- Symmetric lookup: `Fire + Air` and `Air + Fire` both resolve to `'storm'`
- Outcomes include standalone combo elements (e.g. `'storm'`, `'glacier'`, `'magma'`, `'plasma'`, `'mist'`, `'aurora'`) and base-element stabilities for same-element fusions
- Light+Darkness maps to `'aether'` (the 5% special Aether / unstable-Void outcome described in T5.2 is implemented separately in mission logic)

### 2. Exported helper functions

- **`getFusionResult(elementA, elementB)`** — Normalizes and looks up the fused element from the matrix
- **`getFusionMatrix()`** — Returns a shallow copy of the matrix for safe external iteration
- **`getAllPairKeys()`** — Enumerates all known pair keys (useful for UI dropdowns and validation)
- **`BaseElement`** type — Re-exports the constrained 10-element union from `constants.ts`

### 3. Notable pair rules included

| Element A | Element B | Result |
|-----------|-----------|--------|
| fire | air | storm |
| water | ice | glacier |
| fire | ice | steam |
| water | lightning | storm |
| light | darkness | aether |
| earth | nature | overgrowth |
| iron | nature | golem |
| nature | light | life |

Full coverage spans all 55 unique unordered pairs of the 10 base elements.
