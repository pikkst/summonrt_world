# Task T5.2 - Implement Light+Darkness Fusion Special Logic

## What was found

The fusion system in `src/data/fusionMatrix.ts` had a basic matrix lookup for Light+Darkness → Aether, but the special 5% chance mechanics described in the GDD were not implemented. The existing `breedCreatures` function in `src/stores/game/modules/missionModule.ts` did not handle the special case where:

- Light + Darkness fusion should have a **5% chance to produce Aether** (a powerful rare element)
- **95% chance to produce an Unstable Void creature** (with void element, high volatility)

## What was done

### 1. Modified `src/data/fusionMatrix.ts`

- Updated `getFusionResult()` function to accept an optional RNG parameter for testability
- Implemented special logic for Light+Darkness fusion with 5% Aether chance
- Added `isLightDarknessFusion()` helper function to detect this special combination
- Created `UNSTABLE_VOID_CREATURE` template with:
  - Element: void
  - Class: rare (upgraded to epic/legendary based on parent classes when fused)
  - Type: demon
  - Special skill: "Void Blast" (power 25, cost 8)
  - Stats: HP 45, ATK 12, DEF 8, SPD 7, MANA 35

### 2. Modified `src/stores/game/modules/missionModule.ts`

- Added import for `getFusionResult`, `isLightDarknessFusion`, and `UNSTABLE_VOID_CREATURE`
- Enhanced `breedCreatures()` function to:
  - Detect Light+Darkness parent creatures
  - Apply special fusion result (Aether or Unstable Void)
  - Handle special naming: "Aether Manifestation" or "Unstable Void"
  - Apply special mutation: "Unstable Core" for failing the Aether roll
  - Upgrade creature class appropriately: Aether results get a higher class tier

### 3. Created `src/test/fusion.test.ts`

- Tests for standard fusion matrix operations
- Tests for Light+Darkness special fusion with deterministic RNG
- Tests for 5% Aether chance validation
- Tests for `isLightDarknessFusion()` helper
- Tests for `UNSTABLE_VOID_CREATURE` template properties

### 4. Updated `src/data/index.ts`

- Added exports for new fusion functions and the Unstable Void template

## Summary

The Light+Darkness fusion now follows the GDD specification:
- **5% chance**: Produces an Aether creature with enhanced power and rare element
- **95% chance**: Produces an Unstable Void creature with void element and special traits

This creates a risk/reward dynamic where players may attempt risky fusions for a chance at extremely powerful Aether creatures, while risking an unstable void entity that is harder to control.
