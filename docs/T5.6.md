# Task T5.6 – Add Soul Crystal tier-matching (consumed matching highest creature tier)

## What was found
The existing fusion system in `src/stores/game/modules/missionModule.ts` only required 5 Essence to perform Soul Fusion, but according to the GDD specification:

> "Fusion consumes a **Soul Crystal** (tier-matched to highest creature)."

Additionally, only two Soul Crystal variants were defined in the UI (`soul_crystal_common` and `soul_crystal_rare`), but no logic existed to consume them during fusion.

## What was done

### 1. Added Soul Crystal tier variants to `src/ui/GameShell.tsx` (lines ~1507-1512)
- Added `soul_crystal_uncommon` (rarity 2)
- Added `soul_crystal_epic` (rarity 4)
- Added `soul_crystal_legendary` (rarity 5)
- Added `soul_crystal_mythical` (rarity 6)

Now all 6 creature class tiers have matching Soul Crystal variants.

### 2. Added helper function `getSoulCrystalTierForClass` to `src/data/constants.ts` (lines 52-62)
- Maps creature class directly to required Soul Crystal tier:
  - `common` → `soul_crystal_common`
  - `uncommon` → `soul_crystal_uncommon`
  - `rare` → `soul_crystal_rare`
  - `epic` → `soul_crystal_epic`
  - `legendary` → `soul_crystal_legendary`
  - `mythical` → `soul_crystal_mythical`

### 3. Updated `src/stores/game/modules/missionModule.ts` `breedCreatures` function
- Imported `getSoulCrystalTierForClass` from constants
- After validating 5 Essence requirement, determines the effective base class from both parent creatures (choosing the higher-tier class when they differ)
- Calculates the required Soul Crystal tier based on the highest creature class
- Searches player inventory for the matching Soul Crystal (`soul_crystal_{tier}`)
- Returns early with warning message if required Soul Crystal is not found
- Consumes the Soul Crystal from inventory upon successful fusion (in addition to the Essence consumption)

### 4. Fusion Requirements Summary
Soul Fusion now requires:
1. **Location**: Must be at a City tile (Soul Forge)
2. **Essence**: 5 units
3. **Soul Crystal**: Tier-matching to the highest parent creature class

| Creature Class | Required Soul Crystal |
|----------------|----------------------|
| common         | Soul Crystal (Common)   |
| uncommon       | Soul Crystal (Uncommon) |
| rare           | Soul Crystal (Rare)     |
| epic           | Soul Crystal (Epic)     |
| legendary      | Soul Crystal (Legendary) |
| mythical       | Soul Crystal (Mythical) |
