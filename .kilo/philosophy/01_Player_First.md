# 01 - Player First

## Principle

The player journey is the center of SummonerWorld.

Every system should answer:

```text
What meaningful choice does this give the player?
```

## Player Value

A system should help the player:

- become stronger
- make better decisions
- command creatures
- shape worlds
- build identity
- manage risk
- gain reputation
- create history
- progress toward long-term goals

## System Direction

Creatures, dungeons, NPCs, worlds, economy, quests, and housing all exist to support the player journey.

They may be simulated deeply, but simulation depth must still connect back to player agency.

## AI Rule

Do not implement a system only because it is technically interesting.

Implement it because it improves the player's long-term journey.
