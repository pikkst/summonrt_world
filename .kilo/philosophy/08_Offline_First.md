# 08 - Offline First

## Principle

Core gameplay must work without internet access.

SummonerWorld begins as an offline-first browser RPG and later grows toward online/MMO infrastructure.

## Offline Core

Offline core systems include:

- player progression
- creature management
- dungeon progression
- combat
- crafting
- travel
- save/load
- world simulation
- reports

## Design Meaning

Online features may enhance the game later, but they must not become required for core play.

## AI Rule

Do not add network dependency to core gameplay unless the task explicitly requires an online-only feature.
