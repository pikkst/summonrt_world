# Implement Task

Use this command to implement one planned SummonerWorld task.

Read `.kilo/prompts/01_implement_task.md` for the full workflow.

Required steps:
1. Read `SummonerWorld_Tasks.md` for the current task.
2. Read relevant `.kilo/rules` files.
3. Search for existing implementation.
4. Create branch `feature/[TASK-ID]-short-name`.
5. Implement the task.
6. Add or update tests.
7. Run validation.
8. Commit and push.
