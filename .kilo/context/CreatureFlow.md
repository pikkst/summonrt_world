# Creature Flow

## Lifecycle

```text
Discovery
  -> Encounter
  -> Capture / Contract
  -> Summoning
  -> Commands
  -> Training
  -> Leveling
  -> Mutation
  -> Evolution
  -> Fusion
  -> Genetics / Lineage
  -> Memory / Legacy
```

## Contract Direction

Creature ownership should move toward contracts.

A contract can later support:

- bond
- trust
- loyalty
- permissions
- trade status
- PvP eligibility
- fusion rights

## System Connections

Creatures connect to:

- player progression
- combat
- dungeon utility
- world ecology
- economy
- crafting
- quests
- housing

## Rule

Do not make creatures replace the player as root architecture.

Creatures are central to gameplay, but the player journey remains the root.
