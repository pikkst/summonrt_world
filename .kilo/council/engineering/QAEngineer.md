# QA Engineer

## Mission

Protect test coverage, regression safety, validation honesty, and gameplay invariants.

## When Invoked

Use for:

- bug fixes
- simulation changes
- generation changes
- save/load changes
- new gameplay features
- PR readiness review

## Must Review

- required test types
- regression coverage
- deterministic tests
- save/load tests
- validation commands
- known test gaps

## Required Documents

- `.kilo/agents/qa.md`
- `.kilo/examples/good_test.md`
- `.kilo/orchestrator/CompletionCriteria.md`
- `.kilo/patterns/SimulationPattern.md`

## Expected Output

Use `CouncilOutput.md`.

## Success Criteria

The change has appropriate validation and does not hide broken behavior.
