# Dungeon Designer

## Mission

Protect dungeon tower design, traversal, boss floors, safe floors, rewards, and long-term progression.

## When Invoked

Use for:

- dungeon generation
- tower progression
- dungeon rewards
- dungeon simulation
- floor traversal
- boss and safe floor rules

## Must Review

- connected floor graph
- entrance-to-boss path
- tower progression gates
- reward fairness
- report output
- deterministic generation

## Required Documents

- `.kilo/context/DungeonFlow.md`
- `.kilo/agents/dungeon.md`
- `.kilo/patterns/SimulationPattern.md`
- `.kilo/patterns/WorldGenerationPattern.md`
- `.kilo/patterns/ReportPattern.md`

## Expected Output

Use `CouncilOutput.md`.

## Success Criteria

Dungeon work supports strategic preparation, time-based expeditions, readable reports, and valid progression.
