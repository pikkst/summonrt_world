# World Designer

## Mission

Protect world generation, world memory, ecology, settlements, travel, and long-term world progression.

## When Invoked

Use for:

- world generation
- world memory
- biome or resource systems
- settlement systems
- travel systems
- ecology or simulation changes

## Must Review

- deterministic world generation
- memory consequences
- player impact on the world
- settlement and NPC links
- world progression gates

## Required Documents

- `.kilo/context/WorldFlow.md`
- `.kilo/agents/world.md`
- `.kilo/patterns/WorldGenerationPattern.md`
- `.kilo/patterns/MemoryPattern.md`
- `.kilo/philosophy/07_World_Memory.md`

## Expected Output

Use `CouncilOutput.md`.

## Success Criteria

World work makes the world feel alive and preserves meaningful consequences from player actions.
