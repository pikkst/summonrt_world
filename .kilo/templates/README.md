# Kilo Templates

This folder contains reusable templates for Kilo Code and other AI coding agents working on SummonerWorld.

Templates should be copied and filled in when creating feature specs, tasks, bug reports, architecture decisions, system specs, PRs, test plans, refactor plans, release notes, or changelog entries.

## Template Index

- `FEATURE_SPEC.md` - feature design and implementation scope.
- `TASK_TEMPLATE.md` - single implementation task definition.
- `BUG_REPORT.md` - bug report format.
- `ADR_TEMPLATE.md` - architecture decision record.
- `SYSTEM_SPEC.md` - system-level specification.
- `PR_TEMPLATE.md` - Pull Request body template.
- `TEST_PLAN.md` - test planning template.
- `DESIGN_DECISION.md` - gameplay/design decision template.
- `REFACTOR_PLAN.md` - refactor planning template.
- `RELEASE_NOTES.md` - release or sprint notes template.
- `CHANGELOG_ENTRY.md` - changelog entry template.

## Usage Rule

Do not fill templates with invented completion claims.

If something was not validated, write that it was not validated.
